FTT - Blockchain of blockchains


FTT is a Private Blockchain run by FinTechToken on Amazon Web Services (AWS). FinTechToken works just like Ethereum except it is totally free so you can try it without risking anything.

FTT is on block {{!blocknumber ? 0 : blocknumber | number}}. We show the last 3 digits in the navigation bar at the top of your screen.

Transactions happen when a new block is mined, which should happen approximately every second.
